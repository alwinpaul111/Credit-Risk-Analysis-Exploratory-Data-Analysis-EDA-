Credit Risk Analysis – Exploratory Data Analysis (EDA)
📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on credit applicant data to identify patterns and factors influencing loan default risk. Financial institutions use such analysis to assess borrower profiles and make data-driven credit approval decisions.

The analysis explores relationships between variables such as income, employment length, loan amount, and credit history to identify potential indicators of default risk.

Key Insights

From the analysis, several patterns were identified:

Applicants with lower income levels showed a higher probability of loan default.

Higher loan-to-income ratios were associated with increased credit risk.

Applicants with short employment history were more likely to default.

Certain loan purposes and interest rates showed stronger correlations with default behavior.

These insights can help financial institutions improve credit risk assessment models.
